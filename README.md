# Cars24_Price_Prediction
Cars24 Price Prediction
🚗 Used Car Price Prediction Dashboard

An interactive Used Car Price Prediction Web App built using Python, Streamlit, Pandas, and Scikit-learn.
This application allows users to estimate the market price of a car based on key features like brand, model, year, fuel type, kilometers driven, and transmission.

🚀 Live Application

🔗 Cars24 Price Prediction

📊 Features

🚗 Price Estimation – Predicts car price based on user inputs.

📝 Feature Inputs – Select car make, model, year, fuel type, transmission, and kilometers driven.

📊 Interactive Dashboard – User-friendly interface using Streamlit.

💡 Data-driven Insights – Leverages machine learning model trained on historical car pricing data.
